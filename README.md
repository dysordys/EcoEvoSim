# EcoEvoSim



## Installation

To install the EcoEvoSim, use the Julia package manager. You can add it to your Julia environment by running:

```julia
using Pkg
Pkg.add("EcoEvoSim")
```



## Usage

Once installed, you can use EcoEvoSim by importing it in your Julia code:

```julia
using EcoEvoSim
```
